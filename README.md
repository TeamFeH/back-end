# back-end
Back-end of the project

##Install...

pip install -r requirements.pip

##... & Launch

cd back_end
./manage.py migrate
./manage.py runserver
go to http://127.0.0.1:8000/

##Using the admin

cd back_end
./manage.py createsuperuser
./manage.py runserver
go to http://127.0.0.1:8000/admin
